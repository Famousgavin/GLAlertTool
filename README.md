## 二、安装
1. [CocoaPods](https://cocoapods.org/)安装：
```
pod 'GLAlertTool'

如果搜索不到，请先运行   rm ~/Library/Caches/CocoaPods/search_index.json  在搜索。

```

2. 下载ZIP包,将 `GLAlertTool` 资源文件拖到工程中。
